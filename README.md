# CRUD Web App

This project is a CRUD Web Application developed using:

- React JS (Frontend)
- Laravel API (Backend)
- MySQL Database

The application allows admin users to:
- Add products
- Edit products
- Delete products

Normal users can:
- View products only

---

# Technologies Used

## Frontend
- React JS
- Axios
- React Router DOM
- CSS

## Backend
- Laravel
- Sanctum Authentication
- REST API
- MVC Architecture

## Database
- MySQL
